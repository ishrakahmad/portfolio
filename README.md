## ⚙ Deployment

This project is automatically deployed through **Vercel** whenever new changes are pushed to the **main** branch.

Deployment Workflow:

```
VS Code
   ↓
Git Commit
   ↓
GitHub Repository
   ↓
Vercel Deployment
   ↓
Custom Domain (ishrakahmad.me)
```
